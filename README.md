# Description
This repository houses files used in the analysis of the Canadian Urban Forest Census (2025).

## Urban Census Subdivisions 
File Name: urban_csds.gpkg (layer = 'urban_csds')

This geopackage includes polygons of the 343 census subdivisions meeting the definition of urban used by Statistics Canada:

> Urban areas are those continuously built-up areas having a minimum population concentration of 1,000 persons and a population density of at least 400 persons per square kilometer based on the previous census.
